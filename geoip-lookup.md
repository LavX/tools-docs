# GeoIP Lookup

The **GeoIP Lookup** tool provides geolocation and network information for any IPv4 or IPv6 address. It helps developers, security researchers, and network administrators identify the approximate physical location and service provider associated with an IP address.

## üöÄ Key Features

- **Geographic Data**: Retrieve continent, country, region, and city information.
- **Network Insights**: Identify the Internet Service Provider (ISP) handling the IP address.
- **IPv4 & IPv6 Support**: Full compatibility with both legacy and modern IP address standards.
- **Private IP Detection**: Automatically identifies private/local IP ranges (e.g., 192.168.x.x, 10.x.x.x) and provides appropriate warnings.
- **Interactive Map**: Visualizes the approximate location using an embedded OpenStreetMap view for public IP addresses.
- **Deep Linking**: Support for URL parameters allows you to share or bookmark specific lookups (e.g., `?ip=8.8.8.8`).
- **One-Click Copy**: Quickly copy the IP address to your clipboard for use in other tools or documentation.

## üõ†Ô∏è Technical Details

- **Data Provider**: Powered by [ip-api.com](https://ip-api.com) for reliable geolocation data.
- **Privacy First**: The tool does not store or persist any IP addresses or lookup history. All processing is done in real-time.
- **Validation**: Built-in validation ensures that only correctly formatted IPv4 and IPv6 addresses are processed.
- **Approximate Accuracy**: Geolocation data is based on IP databases and should be considered approximate. Accuracy varies by provider and region.

## üìñ How to Use

1. **Enter IP Address**: Type or paste an IPv4 or IPv6 address into the search field.
2. **Quick Examples**: Use the provided example buttons (e.g., `8.8.8.8`) to see the tool in action instantly.
3. **Analyze Results**:
    - **IP Info**: View the address type and whether it is a public or private IP.
    - **Map View**: See the approximate geographic location on the interactive map.
    - **Location Details**: Review the country (with flag emoji), city, and timezone.
    - **Network Details**: Check the ISP information.
4. **Copy & Share**: Use the "Copy" button to grab the IP, or share the URL with the `ip` parameter to show the results to others.
