## üìñ How to Use

1. **Access the Tool**: Navigate to the [RegEx Tester](https://tools.lavx.hu/tools/regex-tester) page.
2. **Enter Pattern**: Type your regular expression in the pattern input field. The tool provides automatic syntax validation.
3. **Configure Options**: Toggle regex flags (global, ignore case, multiline, etc.) using the provided switches.
4. **Load Examples**: Click any of the built-in example buttons to quickly load common patterns like email or URL validation.
5. **Test Your Pattern**: Enter your test text in the "Test String" area and click the **"Test Regex"** button to execute the search.
6. **View Results**: Examine highlighted matches and detailed match information in the results panels
7. **Manage History**: Access previously tested patterns from the history section for quick reuse
8. **Learn Common Patterns**: Explore the regex reference guide with common patterns and special characters
