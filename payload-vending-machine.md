## üìã Available Payload Categories

### SQL Injection (SQLi)
Authentication bypasses, UNION-based queries, blind injection, time-based delays, error-based exploits, and polyglot payloads for MySQL, MSSQL, and PostgreSQL.

### Cross-Site Scripting (XSS) 
Basic alert payloads, image onerror handlers, SVG exploits, XSS polyglots, Angular/Vue template injections, and protocol handlers.

### Remote Code Execution (RCE)
Unix command injection, Windows RCE, reverse shells, blind command execution, and various command separators.

### Local File Inclusion (LFI)
Directory traversal paths, null byte bypasses, double-encoding, Windows targets, and PHP wrapper exploits.

### Server-Side Template Injection (SSTI)
Jinja2, Twig, and Java Spring template injection payloads for various frameworks.

### XML External Entity (XXE)
File read exploits, SSRF attacks, blind out-of-band exfiltration, and denial of service payloads.

### Miscellaneous
LDAP injection, XPath injection, and format string vulnerabilities.
