## üîß Advanced Features

### Visual Query Builder
- **Block-Based Construction**: Visually construct queries by adding operator and keyword blocks
- **Drag & Drop Reordering**: Easily change the sequence of query components using a drag-and-drop interface

### Comprehensive Query Library
The tool includes 12 specialized categories:

- **Industrial Control Systems**: SCADA, PLCs, Modbus, BACnet, and industrial protocols
- **IoT Devices**: Smart devices, cameras, routers, printers, and connected appliances
- **Network Infrastructure**: Cisco, Juniper, routers, switches, and network equipment
- **Exposed Databases**: MongoDB, Elasticsearch, Redis, MySQL, and other databases
- **Default Credentials**: Systems with default passwords or weak authentication
- **Vulnerable Services**: Known vulnerable versions and CVE-affected services
- **Cloud Misconfigurations**: Exposed S3 buckets, APIs, and cloud service misconfigurations
- **Web Servers**: Apache, nginx, IIS, and web server fingerprinting
- **Remote Access**: RDP, VNC, SSH, Telnet, and remote management interfaces
- **Webcams & Cameras**: IP cameras, RTSP streams, DVR/NVR systems
- **Healthcare**: DICOM servers, HL7 endpoints, and medical devices
- **Energy & Utilities**: Smart meters, power grids, solar inverters, and utilities

### Target Filtering Options
Specify search targets by:
- **IP Address**: Single IP or IP range
- **Domain Name**: Hostname or domain
- **Organization**: Company or organization name
- **Network**: CIDR notation for network ranges
