## üõ†Ô∏è Key Features

### Case Conversion
- **UPPERCASE**: Convert text to all capital letters
- **lowercase**: Convert text to all lowercase letters
- **Capitalize Each Word**: Capitalize the first letter of each word
- **Sentence case**: Capitalize the first letter of each sentence
- **iNVERT cASE**: Invert the case of each character
- **AlTeRnAtInG cAsE**: Alternate between uppercase and lowercase letters

### Text Transformation
- **Reverse Text**: Reverse the order of characters
- **Remove Spaces**: Remove all spaces from text
- **Remove Line Breaks**: Replace line breaks with spaces
- **Trim Whitespace**: Remove leading and trailing spaces
- **Remove Extra Spaces**: Replace multiple spaces with single spaces
- **Remove Punctuation**: Remove all punctuation marks
- **ROT13 Cipher**: Apply ROT13 encoding (A‚ÜíN, B‚ÜíO, etc.)
- **Atbash Cipher**: Apply Atbash cipher (A‚ÜîZ, B‚ÜîY, etc.)
- **Numeronym Generator**: Convert words to numeronyms (e.g., "Internationalization" ‚Üí "i18n")
- **To Hexadecimal**: Convert text to hexadecimal representation
- **From Hexadecimal**: Convert hexadecimal back to text
- **To Char Codes (CSV)**: Convert text to character code values
- **From Char Codes (CSV)**: Convert character codes back to text
- **To Binary String**: Convert text to 8-bit binary representation
- **From Binary String**: Convert binary string back to text

### Text Formatting
- **Add Line Numbers**: Prepend line numbers to each line
- **Sort Lines (A-Z)**: Sort lines alphabetically
- **Sort Lines (Z-A)**: Sort lines in reverse alphabetical order
- **Shuffle Lines**: Randomly shuffle the order of lines
- **Join Lines with Comma**: Replace line breaks with a comma and space

### Text Analysis
*Note: These operations append the statistics to the end of your text.*
- **Count Characters**: Append the total character count
- **Count Words**: Append the total word count
- **Count Lines**: Append the total line count
- **Count Paragraphs**: Append the total paragraph count

### Encoding/Decoding
- **URL Encode**: Convert text to URL-encoded format
- **URL Decode**: URL-decode text back to original form
- **HTML Encode**: Convert special characters to HTML entities
- **HTML Decode**: Convert HTML entities back to characters

### Find & Replace
- **Replace Text**: Simple find and replace operation
- **Replace with Regex**: Find and replace using regular expressions with custom flags
