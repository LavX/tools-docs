## üìñ How to Use

### 1. **Access the Tool**
Navigate to the [OSINT Dork Builder](https://tools.lavx.hu/tools/osint-dork-builder) page.

### 2. **Choose Your Mode**
Select from three powerful modes:
- **PreBuilt Library**: Browse and search existing dorks
- **Visual Builder**: Create custom dorks with the visual interface
- **AI Assistant**: Use AI to generate, explain, optimize, or suggest dorks

### 3. **Set Target Domain (Optional)**
Enter a target domain to automatically include `site:` operator in all your queries. The tool validates domain format automatically.

### 4. **Build or Select Dorks**
- **PreBuilt Mode**: Search, filter by category, browse paginated results
- **Builder Mode**: Click operators, file types, keywords, or use templates
- **AI Mode**: Describe what you want to find or paste existing dorks for analysis

### 5. **Execute or Export**
- **Copy**: Copy queries to clipboard with one click
- **Execute**: Launch searches directly in your chosen search engine
- **Use**: Send queries to the main query panel for further editing
