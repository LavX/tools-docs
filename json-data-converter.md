## üìñ How to Use

1. **Access the Tool**: Navigate to the [JSON Data Converter](https://tools.lavx.hu/tools/json-data-converter) page.
2. **Select Input Format**: Choose your input format from the dropdown or leave it on **Auto-Detect**.
3. **Select Output Format**: Choose the desired target format (JSON, YAML, XML, TOML, or CSV).
4. **Input Your Data**: Paste your source data into the "Input Data" textarea.
5. **View Results**: The converted data will appear instantly in the "Output Data" area.
6. **Copy/Clear**: Use the "Copy Output" button to copy the result or "Clear" to start over.

### Format Specifics
- **TOML**: Note that TOML has strict structure requirements (e.g., no mixed-type arrays).
- **CSV**: Conversion to CSV requires the input data to be an array of objects. For best results with CSV input, manual format selection is recommended.
- **XML**: Non-object inputs are automatically wrapped in a `<root>` tag.
