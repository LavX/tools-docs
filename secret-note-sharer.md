# Secret Note Sharer

Create and share one-time readable, password-protected, encrypted notes with optional email notifications when viewed.

## üöÄ Why Use Secret Note Sharer?

Secret Note Sharer provides a secure method for sharing sensitive information that can only be read once. Perfect for sharing passwords, API keys, or confidential messages that you want to ensure are seen only by the intended recipient and then permanently destroyed.

### Security First
- **Strong Encryption**: Uses AES-256-GCM encryption with PBKDF2 key derivation (100,000 iterations)
- **Client-Side Validation**: Password strength is enforced with minimum 8-character requirement
- **One-Time Access**: Notes are permanently deleted after being viewed once
- **Secure Password Transmission**: Password is sent in the request body, not as a URL parameter

## üìñ How to Use

### Creating a Secret Note
1. **Access the Tool**: Navigate to the [Secret Note Sharer](https://tools.lavx.hu/tools/secret-note-sharer) page.
2. **Enter Your Secret**: Type your confidential message in the text area.
3. **Set a Password**: Create a strong password (minimum 8 characters) to protect your note.
4. **Enable Notifications** (Optional): Toggle the notification switch and enter your email to be notified when the note is viewed.
5. **Customize Notification** (Optional): Add a custom message to be included in the notification email.
6. **Create Note**: Click "Create Secret Note" to generate your encrypted note.
7. **Copy the Link**: Copy the generated shareable link immediately - you cannot retrieve it later.

### Viewing a Secret Note
1. **Access the Link**: Use the shareable link provided by the sender.
2. **Enter Password**: Input the password required to decrypt the note.
3. **View Note**: The decrypted content will be displayed.
4. **Note Destruction**: The note is automatically deleted after viewing and cannot be accessed again.
