### Advanced Task Management
- **Create Tasks**: Quickly add new tasks with a simple input form
- **Edit Tasks**: Modify existing task text inline
- **Complete Tasks**: Toggle completion status with a beautiful switch control
- **Delete Tasks**: Remove completed or unwanted tasks
- **Smart Shortcuts**: Use Enter to add tasks, Enter/Escape to save/cancel edits

### Priority System
- **Three Priority Levels**: Low, Medium, High
- **Color-Coded Indicators**: Visual priority indicators with distinct colors
  - **Low Priority**: Green background with green text
  - **Medium Priority**: Orange background with orange text  
  - **High Priority**: Red background with red text
- **Priority Assignment**: Select priority when creating each task

### Category System
- **Built-in Categories**: Personal, Work, Shopping, Other
- **Custom Categories**: Add unlimited custom categories with "+ New" button
- **Category Organization**: Each task can be assigned to any category
- **Category Persistence**: Categories are saved and restored across sessions

### Data Persistence & Privacy
- **Automatic Saving**: All tasks are automatically saved to browser localStorage
- **Session Restoration**: Tasks and categories are restored when you return
- **Privacy-Focused**: No account registration or cloud storage required
- **Offline-Ready**: Works without internet connection once loaded

### User Experience
- **Clean Interface**: Modern, gradient-based design with smooth animations
- **Visual Status**: Completed tasks show with reduced opacity and strikethrough text
- **Timestamps**: See task creation dates and completion dates
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices

### Task Display Features
- **Priority Badges**: Each task shows its priority level as a colored badge
- **Category Badges**: Tasks display their assigned category
- **Date Information**: Creation date shown for each task

## üìñ How to Use

1. **Access the Tool**: Navigate to the [To-Do List](https://tools.lavx.hu/tools/todo-list) page.

2. **Add Your First Task**:
   - Enter your task description in the input field
   - Select a priority level (Low, Medium, or High)
   - Choose a category or create a new one
   - Press Enter or click "Add Task" button

3. **Manage Your Tasks**:
   - Toggle completion status using the switch control on each task
   - Click "Edit" to modify task text inline
   - Click "Delete" to remove unwanted tasks
   - Completed tasks show with reduced opacity

4. **Work with Categories**:
   - Use the category dropdown to assign tasks to existing categories
   - Click "+ New" to add custom categories
   - All categories are preserved across browser sessions

5. **Track Progress**:
   - Monitor completion status with visual indicators
   - View task creation and completion timestamps
   - Tasks are automatically sorted and filtered in the background

6. **Privacy & Data**:
   - All tasks and categories are stored locally in your browser
   - No account or registration required
   - Data persists even when browser is closed and reopened
   - Works offline once the page is loaded
