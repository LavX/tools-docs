## üîß URL Components Explained

The tool breaks down URLs into these standard components:
- **Protocol**: The scheme (http, https, ftp, etc.)
- **Username/Password**: Authentication credentials when present
- **Hostname**: The domain name (e.g., example.com)
- **Port**: The port number if specified
- **Pathname**: The path after the domain (e.g., /path/to/page)
- **Search**: The query string with parameters
- **Hash**: The fragment identifier (everything after #)
