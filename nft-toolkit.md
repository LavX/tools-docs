## üìñ How to Use

### Metadata Viewer
1. **Select Tool**: Navigate to the [NFT Toolkit](https://tools.lavx.hu/tools/nft-toolkit) and click the "Metadata Viewer" tab
2. **Enter NFT Details**: Input either:
   - A supported collection name (e.g., "BAYC", "Azuki", "Doodles")
   - A valid Ethereum contract address (0x...)
3. **Specify Token ID**: Enter the specific token number
4. **Fetch Data**: Click "Fetch Metadata" to retrieve the NFT information
5. **View Results**: Examine the metadata including name, description, image, and attributes

### Floor Price Tracker
1. **Select Tool**: Navigate to the [NFT Toolkit](https://tools.lavx.hu/tools/nft-toolkit) and click the "Floor Price Tracker" tab
2. **View Live Data**: The tool automatically displays current floor prices
3. **Monitor Changes**: Track 24-hour price changes and trends
4. **Refresh Data**: Use the refresh button to update prices manually
