## üî¨ Understanding the Results

### Option Pricing Results
- **Option Price**: The theoretical fair value calculated using Black-Scholes
- **Standard Contract Value**: Multiplies by 100 for typical option contracts
- **Intrinsic Value**: Immediate exercise value (minimum of 0)
- **Time Value**: Additional premium based on time remaining and volatility

### Option Greeks
- **Delta (Œî)**: Sensitivity to underlying price changes
- **Gamma (Œì)**: Rate of change in Delta
- **Theta (Œò)**: Daily time decay of option value
- **Vega (V)**: Sensitivity to volatility changes
- **Rho (œÅ)**: Sensitivity to interest rate changes

### Profit/Loss Analysis
- **Break-Even Point**: Stock price needed for option to become profitable
- **Maximum Profit**: Best-case scenario potential return
- **Maximum Loss**: Limited to premium paid for long options
- **ROI Scenarios**: Return on investment calculations for various outcomes
