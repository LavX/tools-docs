## üîß Advanced Features

### Input Validation
- Validates timestamp ranges (supports ~1000 AD to ~3000 AD)
- Auto-detects seconds vs milliseconds format
- Provides clear error messages for invalid inputs

### Dynamic Timezone Handling
- Calculates current timezone offsets including DST
- Updates timezone abbreviations automatically
- Supports IANA timezone identifiers

### Output Format Options
The tool supports 10 different output formats:
1. **ISO 8601 (Local)**: `2024-12-14T01:27:00+01:00`
2. **ISO 8601 (UTC)**: `2024-12-14T00:27:00.000Z`
3. **RFC 2822**: `Sat, 14 Dec 2024 01:27:00 +0100`
4. **Unix (Seconds)**: `1734134820`
5. **Unix (Milliseconds)**: `1734134820000`
6. **Relative Time**: `3 hours ago`
7. **Long Format**: `Saturday, December 14, 2024 at 1:27:00 AM`
8. **Short Format**: `12/14/2024, 1:27 AM`
9. **Date Only**: `2024-12-14`
10. **Time Only**: `01:27:00`
