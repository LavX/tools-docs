# IPv4 Converter

Convert IPv4 addresses between Dotted Decimal, Decimal (Integer), Hexadecimal, and Binary formats with real-time validation and conversion.

## üöÄ Why Use IPv4 Converter?

Networking tasks often require switching between different representations of an IP address. Whether you're debugging firewall rules, calculating subnet masks, or working with low-level network protocols, the **IPv4 Converter** provides a fast and reliable way to translate addresses into the format you need.

### Developer-Centric Features
- **Real-Time Conversion**: Results update instantly as you type or change the input format.
- **Flexible Input Formats**: Supports various notations including hex prefixes (`0x`), binary prefixes (`0b`), and underscores in decimal values for better readability.
- **One-Click Copy**: Quickly copy any of the converted formats to your clipboard.
- **Input Validation**: Provides clear error messages if the input format is invalid or out of the IPv4 range (0 to 4,294,967,295).

## üõ†Ô∏è Key Features

- **Multi-Format Support**:
    - **Dotted Decimal**: Standard notation (e.g., `192.168.1.1`). Validates four octets (0-255).
    - **Decimal (Integer)**: 32-bit unsigned integer representation (e.g., `3232235777`).
    - **Hexadecimal**: 8-digit hex representation (e.g., `0xC0A80101`).
    - **Binary**: 32-bit binary representation with octet separation (e.g., `11000000.10101000.00000001.00000001`).
- **Smart Parsing**:
    - **Decimal**: Allows underscores for readability (e.g., `3_232_235_777`).
    - **Hexadecimal**: Accepts both prefixed (`0x...`) and raw strings, up to 8 hex characters.
    - **Binary**: Handles spaces, dots, and `0b` prefixes, automatically padding to 32 bits.
- **Privacy-Focused**: All conversions are performed locally in your browser; no IP data is sent to the server.
