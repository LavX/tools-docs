## üìñ How to Use

1. **Access the Tool**: Navigate to the [UUIDs Generator](https://tools.lavx.hu/tools/uuids-generator) page.
2. **Select UUID Version**: Choose your desired UUID version from the dropdown:
   - **v1**: For timestamp-based UUIDs (can be predictable)
   - **v4**: For random UUIDs (recommended for most use cases)
   - **v5**: For deterministic UUIDs based on namespace and name
3. **Configure Options**:
   - For v1 and v4: No additional input required
   - For v5: Enter a name string and optionally provide a specific namespace UUID (defaults to DNS namespace: `6ba7b810-9dad-11d1-80b4-00c04fd430c8`)
4. **Generate UUID**: Click the "Generate UUID" button
5. **Copy Your UUID**: Use the copy button to save the generated UUID to your clipboard
