## üìñ How to Use

1. **Access the Tool**: Navigate to the [Markdown Editor](https://tools.lavx.hu/tools/markdown-editor) page.
2. **Choose Your Mode**: Select between Markdown or WYSIWYG editing mode
3. **Start Writing**:
   - In Markdown mode: Write in the left editor with Markdown syntax
   - In WYSIWYG mode: Use the toolbar buttons for formatting
4. **Customize Your View**: Select a theme and toggle auto-preview. If auto-preview is off, use the **Render** button to update the preview manually.
5. **Save Your Work**: Use the **Save** button to store the current document in your browser history.
6. **Export**: Use the Export menu to save as HTML or print as PDF.
