## üìñ How to Use

1. **Set Foreground Color**: Enter a HEX code, RGB values, or HSL values for your text.
2. **Set Background Color**: Enter the color values for your background.
3. **Analyze Results**: The tool automatically calculates the contrast ratio and displays pass/fail status for WCAG AA and AAA levels.
4. **Swap Colors**: Use the **Swap Colors** button to see if the reverse combination also meets accessibility standards.
5. **Copy Report**: Click **Copy All Results** to get a formatted summary of the analysis for your documentation or handoff.

## üî¨ Technical Details

The tool calculates the contrast ratio using the relative luminance formula defined in WCAG 2.0:
`Contrast Ratio = (L1 + 0.05) / (L2 + 0.05)`
Where `L1` is the relative luminance of the lighter color and `L2` is the relative luminance of the darker color.
